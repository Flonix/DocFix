Setting up a cross reference

Setting up a cross reference is quite easy. It is nearly the same as defining a link in HTML syntax. Set the href attribute to "#" and add a onclick event.

Setting up a cross reference destination

First of all, a destination can be every html tag used in your documentation. The only thing that matters is the value of the id attribute.

Please read "How cross-referencing works" to make sure your destination is really found in the document. 

	sourcetag 
	...
You may use the css class docfix-hidden to hide your destination to the viewer. If someone searches for your destination or just clicks on a cross-reference the document will still be scrolled to the location of your destination.
Do not use the css class hidden for hiding your destination tags, because this will crash the scrolling.
