Comments

DocFix supports two different possibilities to add some comments to your documentation. The first one is defined by DocFix itself, the second one comes from the bootstrap css library. You may define your own text and comment styles in your own css files.

The DocFix comment

The DocFix comment is produced by using the blockquote html tag.
