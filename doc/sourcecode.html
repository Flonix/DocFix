Source-Code

Source code with line numbering

Use the following html markup to create a source code block with line numbering enabled. 

System.out.println("Your source code...");

An example of Java source code: 

System.out.println("Hello world!");	// Comment
System.out.println("Hello world!");
System.out.println("Hello world!");
System.out.println("Hello world!");
System.out.println("Hello world!");
System.out.println("Hello world!");

Source code without line numbering

Use the following html markup to create a source code block without line numbering. 

System.out.println("Your source code...");

An example of Java source code: 

System.out.println("Hello world!");	// Comment
System.out.println("Hello world!");
System.out.println("Hello world!");
System.out.println("Hello world!");
System.out.println("Hello world!");

Available languages

The source code highlighting should work for C and friends, Java, Python, Bash, SQL, HTML, XML, CSS, Javascript, and Makefiles. It works passably on Ruby, PHP, VB, and Awk and a decent subset of Perl and Ruby.

You may use the class attribute of the pre tag to define the programming language. The only thing you have to do is adding a class called lang-*, where * is the name of the programming language. The following example will enable the css highlighting profile.

Markup for using css highlighting: 

Your css code...
 
body {
	font-size: 10px;
	border: 1px solid black;
}
